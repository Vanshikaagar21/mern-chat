# Steps to run this project:

1. Create a `.env` file in `/api` and set environment variables (`MONGO_URL` and `JWT_SECRET`).
2. In `/api`, run `npm i` and then `nodemon`.
3. In `/client`, run `yarn` and then `yarn run dev`.
